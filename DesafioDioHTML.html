Estrutura Básica HTML

A estrutura básica do HTML (HyperTextMarkup) é composta inicialmente pela tag "!DOCTYPE HTML", logo após isso implementamos a tag "HTML lang=PT-BR".

Ainda seguindo esta logica encontramos o "Head". Onde este elemento contém informações sobre o documento que não são exibidas na página, porém essenciais para o funcionamento dela.

Agora, chegamos ao elemento "body". Este é o espaço onde todo o conteúdo visível da página é colocado. Aqui você pode adicionar textos, imagens, links, vídeos e muito mais.

Listas Ordenadas e Não Ordenadas

As listas não ordenadas são usadas quando a ordem dos itens não é importante. Elas são representadas pelo elemento "ul", que significa "unordered list". Cada item da lista é definido com o elemento "li", que significa "list item". Os itens em uma lista não ordenada são exibidos com marcadores (bolinhas).

As listas ordenadas são usadas quando a ordem dos itens é bem importante, são representadas pelo elemento "ol, que significa "ordered list". Assim como nas listas não ordenadas, cada item é definido com o elemento "li", os itens em uma lista ordenada são numerados automaticamente pelo navegador

Links no HTML

O elemento "a" (Âncora) é usado para criar links em HTML, possuindo um atributo obrigatório chamado HREF (hypertext reference), que liga o destino do link. O texto entre as tags "a"e "/a" é o texto do link que o usuário verá e clicará.

Exemplo: "a href="www.apredizagem.com /a"
